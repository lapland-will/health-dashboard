# 道到云笔记训练数据导入指南

**目标：** 读取您的有道云笔记中的训练总结和训练日志，用于分析训练模式

---

## 📝 导入方式选择

### 方式一：导出为Markdown/文本文件（推荐）⭐

#### 步骤：

1. **打开有道云笔记应用**
2. **找到训练相关的笔记**，例如：
   - 训练日志
   - 训练总结
   - 比赛记录
   - 成绩追踪

3. **导出笔记**：
   - 选择笔记
   - 点击"导出"或"分享"
   - 选择"Markdown"或"文本"格式
   - 保存到：`Personal/Health/TrainingLogs/`

4. **或者手动复制粘贴**：
   - 打开笔记
   - 全选内容（Cmd+A）
   - 复制（Cmd+C）
   - 在Obsidian中新建笔记粘贴
   - 保存到：`Personal/Health/TrainingLogs/`

### 方式二：使用有道云笔记的导出功能

#### 桌面端：

1. 打开有道云笔记Mac客户端
2. 选择要导出的笔记
3. 右键点击 → "导出"
4. 选择格式（Markdown推荐）
5. 选择保存位置

#### 网页端：

1. 访问：https://note.youdao.com/
2. 登录账号
3. 打开笔记
4. 点击"更多" → "导出"
5. 选择Markdown格式

### 方式三：直接提供笔记路径

如果您知道笔记文件在Mac上的存储位置，请告诉我路径：

```
~/Library/Containers/com.youdao.note.YoudaoNoteMac/Data/
```

## 🎯 我需要的训练数据类型

### 优先级高的数据

1. **自由潜水成绩记录**
   - 日期
   - 深度（米）
   - 闭气时间（秒/分钟）
   - 潜水方式（恒重、恒重配脚蹼、自由沉浸等）
   - 成绩/排名

2. **日常训练日志**
   - 训练日期
   - 训练内容
   - 训练时长
   - 身体感觉
   - 表现评价

3. **准备度相关数据**
   - 睡眠质量
   - 疲劳程度
   - 心情状态
   - HRV数据（如有）

4. **比赛成绩**
   - 比赛日期
   - 比赛名称
   - 成绩
   - 排名
   - 条件（水温、地点等）

### 示例数据格式

**格式1：每日日志**
```markdown
## 2026-01-15 训练日志

**训练内容：**
- 恒重下潜：6次，最大深度60米
- 闭气训练：陆地CO2表，5分钟
- 恢复期：充分休息

**身体感觉：**
- 精神：良好
- 疲劳：轻度
- 睡眠：7小时，质量不错

**表现：**
- 下潜轻松，深度渐进
- 闭气时间稳定
```

**格式2：成绩记录**
```markdown
## 比赛成绩 - 2026年1月

### 2026-01-20 深圳挑战赛
- 项目：恒重无脚蹼 (CNF)
- 深度：80米
- 成绩：成功
- 白牌成绩：82米

### 2026-01-25 上海站
- 项目：恒重配脚蹼 (CWT)
- 深度：102米
- 成绩：成功，打破个人记录
```

**格式3：简短日志**
```
2026-01-15: 训练日，60米x6，感觉不错
2026-01-16: 休息日，瑜伽拉伸
2026-01-17: 训练日，65米x4，有点疲劳
```

## 🔍 我能做什么

### 1. 读取和分析训练数据

导入后，我会：
- ✓ 识别训练日期和内容
- ✓ 提取成绩数据
- ✓ 关联Oura Ring健康数据
- ✓ 分析成绩与健康数据的关系
- ✓ 找出最佳表现的模式

### 2. 发现最佳训练时机

我会分析：
- **哪些准备度分数下训练成绩最好**
- **多少睡眠时间后表现最佳**
- **HRV范围与成绩的关系**
- **训练强度与恢复的最佳平衡**
- **哪些天气/环境条件下表现更好**

### 3. 提供个性化建议

基于分析，我会：
- ✓ 识别您的最佳训练窗口
- ✓ 预测最适合的比赛时间
- ✓ 推荐最优训练强度
- ✓ 提醒休息和恢复时间
